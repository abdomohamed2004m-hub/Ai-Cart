# AI Cart Assistant

مساعد تسوق ذكي مدعوم بالذكاء الاصطناعي يوفر عربة تسوق موحدة ومدمجة عبر منصات ومواقع التجارة الإلكترونية المختلفة.

## 🚀 المميزات الرئيسية (Key Features)

* **عربة تسوق عالمية (Universal Shopping Cart):** إمكانية حفظ وتجميع المنتجات من مواقع تسوق مختلفة في سلة واحدة موحدة تابعة للإضافة.
* **تنظيم ذكي ومقارنة:** واجهة مخصصة لمقارنة المنتجات المختارة وتنظيمها بسهولة دون الحاجة لفتح علامات تبويب متعددة.
* **واجهة جانبية سريعة (Sidebar UI):** تصميم مرن يفتح كشريط جانبي أثناء التصفح لتوفير تجربة مستخدم سلسة وسريعة.
* **الخصوصية والأمان:** يتم حفظ جميع بيانات المنتجات محلياً على جهاز المستخدم دون مشاركتها مع أي جهات خارجية.

## 💻 بيئة التشغيل المتوافقة (Compatibility)

* **أجهزة الكمبيوتر (PC/Laptop):** متوافق مع جميع المتصفحات المبنية على نواة كروميوم (Google Chrome, Microsoft Edge, إلخ).
* **أجهزة الأندرويد (Mobile/Android):** يعمل عبر المتصفحات التي تدعم إضافات الكمبيوتر مثل:
  * Mises Browser
  * Lemur Browser
  * Kiwi Browser

## 🛠️ طريقة التثبيت (Installation Guide)

1. قم بتحميل ملف الإضافة المضغوط `.zip` من قسم الملفات في هذا المستودع.
2. افتح صفحة الإضافات في المتصفح الخاص بك (مثال: `chrome://extensions` أو `mises://extensions`).
3. قم بتفعيل **وضع المطور (Developer Mode)** من أعلى الشاشة.
4. قم بسحب وإفلات ملف الـ `.zip` مباشرة داخل الصفحة، أو اضغط على خيار الرفع `+(from .zip/.crx/.user.js)` واختر الملف المضغوط مباشرة.
